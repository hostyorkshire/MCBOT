How to add StoryBoT

  1. 1 Open the MeshCore mobile app on your phone.
  2. 2 Go to Contacts and tap Add Contact.
  3. 3 Choose Scan QR and point your camera at the code above.
  4. 4 Once added, message StoryBoT — type help to get started!
💡 Can't message the bot? Do an advert flood route in MeshCore — this broadcasts your presence over the mesh so the bot can discover and add you as a contact automatically.

Bot Commands

Message What it does
start / new / begin Start a new adventure (defaults to wasteland sci-fi)
start <genre> Start with a specific genre, e.g. start horror or start 3
genres List available genres with their numbers
1 / 2 / 3 Choose the numbered story option presented
restart / reset Reset your current story and start fresh
help / ? Show the full command reference

Available Genres

🏜️ wasteland (default) ☕ cozy 👻 horror 🪖 mil 😂 comedy

Commands also work with a /, !, or \ prefix (e.g. /start horror).

StoryBot runs over LoRa mesh radio — no internet needed on the device.
